# failure_record

## Installation 

```bash
conda create --name failure_record python=3.10
conda activate failure_record
git clone https://github.com/malihass/failure_record.git
cd failure_record
pip install -r requirements.txt
```

## Quick start

```bash
conda activate failure_record
python basic_example.py
```
